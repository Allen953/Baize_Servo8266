# Baize_ServoDriver_esp8266

这是一块32路舵机驱动板，做舵机机器人非常方便。主控用的esp8266.

电源用的5V的电池，电池直接给舵机供电，经过压降之后产生3.3V电压给esp8266模组供电。

这块板儿主要是用于控制sg90舵机，就是那种蓝色舵机，或者mg90s舵机。

后续会升级，升级后可以控制5-12V的pwm舵机。

这样就可以控制高压的舵机，比如型号为mg996r或mg995的舵机。
